# Qwen CLI Configuration

My personal Qwen CLI setup and configuration.

## Contents

- `settings.json` - Main Qwen CLI settings
- `output-language.md` - Output language preferences
- `skills/` - Custom skills configuration

## Setup

1. Clone this repository
2. Copy configuration to your `.qwen` directory:
   ```bash
   cp settings.json ~/.qwen/
   cp output-language.md ~/.qwen/
   cp -r skills/* ~/.qwen/skills/
   ```
